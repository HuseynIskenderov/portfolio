Internet provider website

This website design for Proxima, a modern internet service provider, features a clean and vibrant interface with a blue-to-purple gradient theme that conveys innovation and reliability.
The homepage includes clear navigation links (Home, Plans, Services, Support, Contact) and a welcoming hero section with a strong headline and a call-to-action button to explore plans.
Popular internet plans are showcased in colorful, gradient-styled cards, making pricing and features easily accessible.

Online banking

This mobile banking app UI is designed with a clean, modular layout and a consistent purple gradient color scheme to enhance visual hierarchy and brand identity.
The interface includes multiple key components: a secure PIN-based login screen with a custom numeric keypad, a dynamic line chart for balance tracking, and a transactional overview
